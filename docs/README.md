# Документація BookCatalog

## 1. Опис проєкту

BookCatalog — навчальний проєкт для демонстрації роботи з Git та GitHub.
Проєкт зберігає список книг у текстовому файлі `books.txt`.

## 2. Структура репозиторію

- `README.md` — короткий опис проєкту.
- `LICENSE` — ліцензія MIT.
- `.gitignore` — список файлів та каталогів, які не відстежуються Git.
- `books.txt` — базовий список книг.
- `docs/README.md` — ця документація.

## 3. Робочий процес (workflow)

1. Створення гілки від `main` для нових змін.
2. Додавання або редагування файлів.
3. Коміти з осмисленими повідомленнями.
4. Відправка гілки на GitHub (`git push`).
5. Створення Pull Request та прив'язка до Issue.
6. Code review та злиття у `main`.
7. Оновлення релізів за потреби.

## 4. Список книг

Поточний список книг зберігається у файлі `books.txt` у корені репозиторію.
